# PickOne

A simple decision-making app built with React and TypeScript. Can't decide between options? Let PickOne choose for you!

## Features

- Add multiple options
- Random selection with animations
- Clean, modern interface
- Mobile responsive

## Installation

```bash
# Clone the repo
git clone https://github.com/therandomuser03/pickone.git

# Install dependencies
cd pickone
npm install
npm install lucide-react framer-motion

# Start development server
npm run dev
```

## Usage

1. Enter your options in the input fields
2. Click "Add Another Option" for more choices
3. Hit "Pick One!" to get a random selection
4. Use "Reset" to start over

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React
